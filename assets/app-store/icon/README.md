# App Icon Guidelines

## Required Sizes

1. App Store Icon (1024 x 1024)
- PNG format
- No alpha channel
- RGB colorspace
- No transparency
- Square shape
- 72 DPI resolution

2. iOS App Icons
- iPhone: 60pt (@2x, @3x) - 120x120, 180x180 pixels
- iPad: 76pt (@2x) - 152x152 pixels
- iPad Pro: 83.5pt (@2x) - 167x167 pixels
- Spotlight: 40pt (@2x, @3x) - 80x80, 120x120 pixels
- Settings: 29pt (@2x, @3x) - 58x58, 87x87 pixels
- Notifications: 20pt (@2x, @3x) - 40x40, 60x60 pixels

## Design Guidelines

1. Visual Design
- Keep it simple and recognizable
- Avoid text unless part of a logo
- Test at small sizes for legibility
- Use strong contrast and bold shapes
- Maintain consistent branding

2. Technical Requirements
- No transparency
- No borders or rounded corners (iOS adds these)
- Fill the entire icon canvas
- Use RGB color space
- Export as PNG files

3. Current Design

Secondary: #343a40 (Dark Gray)
