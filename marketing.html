Marketing Guidelines

Brand assets and marketing specifications for Horse Racer AI

Brand Voice

Tone & Style

  • Professional but accessible
  • Technology-forward but not intimidating
  • Enthusiastic about horse racing
  • Clear about entertainment-only purpose

Key Messages

  • Advanced AI predictions for horse racing enthusiasts
  • Stunning 3D race simulations
  • Entertainment and education focused
  • No real-money betting - pure entertainment

Brand Colors

Color Codes

  • Primary Green: #28a745
  • Dark Gray: #343a40
  • Light Background: #f8f9fa
  • Accent Red: #dc3545

Typography

Fonts

  • Primary: Arial, sans-serif
  • Headers: System font stack
  • UI Elements: System font stack

Social Media

Recommended Image Sizes

  • Instagram Post: 1080 x 1080px
  • Instagram Story: 1080 x 1920px
  • Twitter Post: 1200 x 675px
  • Facebook Post: 1200 x 630px
  • LinkedIn: 1200 x 627px

Hashtags

#HorseRacerAI #HorseRacing #AITechnology #RaceSimulation #HorseRacingApp

App Store Assets

Icon Guidelines

  • App Icon: 1024 x 1024px (PNG)
  • Keep icon simple and recognizable
  • Avoid text in icon
  • Use brand green as primary color

Screenshot Guidelines

  • Show key features in first 2 screenshots
  • Include UI elements in proper context
  • Add brief text overlays explaining features
  • Maintain consistent branding

Legal Requirements

Required Disclaimers

  • "For entertainment purposes only"
  • "No real-money betting or gambling features"
  • "AI predictions are simulations, not guaranteed outcomes"

Image Usage

  • Only use licensed or owned horse racing imagery
  • Avoid trademarked race venue names without permission
  • Credit photographers when required
